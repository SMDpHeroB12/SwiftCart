## 1) What is the difference between `null` and `undefined`?

`undefined` এর মানে হচ্ছে একটি ভেরিয়েবল ডিক্লেয়ার করা হয়েছে, কিন্তু কোনো মান দেওয়া হয়নি।
`null` এর মানে হচ্ছে ইচ্ছাকৃতভাবে “কোনো মান নেই” বোঝানো হয়েছে।

---

## 2) What is the use of the `map()` function in JavaScript? How is it different from `forEach()`?

`map()` ব্যবহার করা হয় একটি অ্যারের প্রতিটি মান পরিবর্তন করে নতুন অ্যারে তৈরি করতে।

`forEach()` শুধু লুপ চালায়, কোনো নতুন অ্যারে রিটার্ন করে না।

---

## 3) What is the difference between `==` and `===`?

`==` শুধু মান তুলনা করে (type convert করে)।
`===` মান এবং type দুইটাই তুলনা করে।

- `==` → loose comparison
- `===` → strict comparison

## 4) What is the significance of async/await in fetching API data?

`async/await` asynchronous কোডকে synchronous এর মতো সহজভাবে লিখতে সাহায্য করে।

এটা API fetch করার সময় কোডকে Clean রাখে, error handle সহজ এবং পড়তে সহজ করে।

## 5) Explain the concept of Scope in JavaScript

Scope নির্ধারণ করে একটি ভেরিয়েবল কোথায় ব্যবহার করা যাবে।

### Global Scope

সব জায়গা থেকে ব্যবহার করা যায়।

### Function Scope

শুধু ফাংশনের ভিতরে কাজ করে।

### Block Scope

`{}` ব্লকের ভিতরে সীমাবদ্ধ থাকে।
